Features:

  • Less than 300 mb of ram at system startup.
  • You can easily install packets using pamac or bauh. Both have support for aur and flatpack. You can also install the snap package support in pamac with this command: sudo pacman -S pamac-all

Bugs:

  • Change the pulseaudio equalizer preset to hear a clearer sound.

Tips:

  • Don't use Chromium Docker or Tor browser if you want the best web browsing experience.
